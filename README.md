# 📊 Productivity Dashboard

A modern **Productivity Dashboard Web Application** designed to help users manage daily tasks, tools, and productivity utilities in one place. The dashboard provides an interactive interface with multiple productivity features accessible from a single platform.

🔗 **Live Demo:**  
https://productivity-dashboard-abhineet.netlify.app/

🔗 **GitHub Repository:**  
https://github.com/AbhineetSingh18/Productivity-Dashboard

---

# 🚀 Features

- 📝 **Task Management**
  - Add, manage, and track tasks
  - Improve daily productivity

- 🧮 **Productivity Utilities**
  - Quick tools to help with daily productivity

- ⏱️ **Time & Focus Tools**
  - Features designed to help manage time and maintain focus

- 🎨 **Modern Dashboard UI**
  - Clean and interactive user interface
  - Card-based layout for easy navigation

- 📱 **Responsive Design**
  - Works across desktop, tablet, and mobile devices

# ⚙️ Installation & Setup

Clone the repository

```bash
git clone https://github.com/AbhineetSingh18/Productivity-Dashboard.git
```

Go into the project directory

```bash
cd Productivity-Dashboard
```

Open the project

```bash
Open index.html in your browser
```

Or run using **VS Code Live Server**.

# 🌐 Deployment

This project is deployed using **Netlify**.

Steps to deploy:

1. Push the project to GitHub
2. Connect the repository with Netlify
3. Deploy the site
